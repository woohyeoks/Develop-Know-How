명령어

```
git init => 깃 생성 .git 폴더 자동 생성 
git config --global user.email "woohyeoks807@naver.com"
git config --global user.name "정우혁"
git add 파일이름 : 커밋에 추가할 파일을 선택 
git commit -m "사이트 설명 추가"
git log	// 커밋 로그 확인
git checkout (commit 버전) : 해당 커밋으로 코드를 되돌린다.
git checkout - : 최신 커밋으로 돌아간다.
git remote add origin https://github.com/woohyeoks/iTshirt.git : 원격 저장소에 커밋 올리기
git push origin master : 원격 저장소에 올리기
git clone https://github.com/woohyeoks/iTshirt.git . : 원격 저장소 커밋 로컬 저장소에 내려 받기
```



용어

```
.git 폴더 => Git으로 생성한 버전들의 정보와 원격저장소 주소 등이 들어 있다. 로컬 저장소라고 부른다.
```
